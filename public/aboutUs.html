This website is part of the Team Project "Computational Creativity in the Rise of Machine Learning: The Case of Generative UI Design" at the University of Mannheim.

We aim to build and train a Neural Network which takes an arbitrary logo in svg-format as input and automatically detects which elements of the logo it should automate and how it should automate them. The output of the model will be the animated logo in svg-format.
Since no data set of animated svg logos exists, we built this website to enable people to easily rate the animated logos generated by our model. We will then use these ratings to provide feedback to the model and improve the quality of the animations.
